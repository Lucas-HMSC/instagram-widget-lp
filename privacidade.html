Política de Privacidade

Última atualização: 26 de abril de 2026

Esta Política de Privacidade descreve como a Lucas Carvalho Serviços de Tecnologia ("nós", "nosso") coleta, usa e protege as informações obtidas através da plataforma Instagram Widget by Luck, em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018) e as políticas da plataforma Meta.

1. Quem somos

O Instagram Widget by Luck é um serviço de API que permite a exibição de feeds do Instagram em sites de terceiros. Operamos como processador de dados em relação às informações fornecidas pela API do Instagram / Meta.

2. Dados que coletamos

2.1 Dados dos clientes (usuários do serviço)

Ao contratar o Instagram Widget by Luck, coletamos:

  • Nome e informações de contato (email)
  • Domínio(s) do site do cliente

2.2 Dados do Instagram (via autorização OAuth)

Quando o cliente autoriza nosso aplicativo via Instagram, recebemos da API da Meta:

  • ID do usuário do Instagram
  • Nome de usuário (username)
  • Token de acesso de longa duração — armazenado de forma segura para buscar o feed
  • Metadados das publicações: ID, legenda, tipo de mídia, URL da mídia, URL permanente e data de publicação

Não coletamos senha, email, telefone, dados banceiros ou qualquer outra informação pessoal do Instagram além dos itens acima.

2.3 Dados técnicos

  • Domínio de origem das requisições à API
  • Logs de acesso (para fins de segurança e diagnóstico)

3. Como usamos os dados

  • Buscar e entregar o feed do Instagram do cliente ao seu site
  • Armazenar em cache (Redis, TTL de 2 horas) para otimizar a performance
  • Renovar automaticamente o token de acesso antes de expirar
  • Validar que as requisições partem de domínios autorizados pelo cliente

Não vendemos, alugamos, compartilhamos ou usamos os dados para fins publicitários.

4. Armazenamento e segurança

Os dados são armazenados em servidores da AWS. Adotamos as seguintes medidas de segurança:

  • Comunicação via HTTPS em todos os endpoints
  • Tokens de acesso armazenados em banco de dados com acesso restrito
  • Isolamento por tenant — cada cliente acessa apenas seus próprios dados
  • Validação de origem (CORS) por domínio cadastrado

5. Retenção de dados

Os dados são mantidos enquanto o contrato de serviço estiver ativo. Após o cancelamento, os dados são excluídos em até 30 dias, salvo obrigação legal de retenção.

6. Seus direitos (LGPD)

Você tem direito a:

  • Confirmar a existência de tratamento dos seus dados
  • Acessar seus dados
  • Corrigir dados incompletos ou desatualizados
  • Solicitar a exclusão dos seus dados
  • Revogar o consentimento a qualquer momento

Para exercer seus direitos, acesse nossa página de exclusão de dados ou entre em contato pelo email privacidade@luck.app.br.

7. Dados de terceiros (Meta / Instagram)

Nosso uso das APIs da Meta está sujeito às Políticas da Plataforma Meta. Não armazenamos dados do Instagram além do necessário para a prestação do serviço.

8. Cookies

Este site não utiliza cookies de rastreamento ou publicidade. Utilizamos apenas o armazenamento estritamente necessário para o funcionamento da API.

9. Alterações nesta política

Podemos atualizar esta política periodicamente. Notificaremos os clientes ativos sobre mudanças relevantes por email. O uso continuado do serviço após a notificação constitui aceite das novas condições.

10. Contato

Dúvidas sobre esta política? Entre em contato:

  • Email: contato@luck.app.br
  • Site: luck.app.br
