Exiba o feed do Instagram
diretamente no seu site

Uma API simples que entrega os posts do Instagram em JSON. Sem SDK, sem iframe, sem layout imposto. Você monta o HTML como quiser.

Exemplo de uso da API:

  'https://luck.app.br/api/v1/widget/feed/SEU_CLIENT_ID?limit=12'
);
const { data } = await res.json();

// data = [{ id, caption, media_type, media_url, permalink, timestamp }, ...]

Como funciona

Três passos. Sem complicação.

1

Conecte a conta

Você cadastra o cliente e envia um link seguro. Ele clica, autoriza o Instagram em segundos, e a conexão fica salva.

2

Configure o domínio

Informe o domínio do site do cliente. Nossa API só responde para origens autorizadas — nenhum outro site consegue usar o token do seu cliente.

3

Consuma o feed

Um fetch() retorna o JSON com os posts. Monte o layout que quiser, sem amarrações de CSS ou HTML impostos pela ferramenta.

Tudo que você precisa

Infraestrutura robusta para você focar no produto.

Cache automático

Redis com TTL de 2 horas. O site do cliente carrega instantaneamente sem bater na API do Instagram a cada visita.

Renovação automática de tokens

Cron job diário que renova os tokens antes de expirarem. Sem intervenção manual, sem perda de acesso para o cliente.

CORS por domínio

Cada cliente tem seus domínios autorizados. A API rejeita requisições de origens não cadastradas automaticamente.

Multi-tenant

Gerencie múltiplos clientes em uma única infraestrutura. Cada um com sua própria conta Instagram e domínios isolados.

JSON puro, zero lock-in

Nenhum HTML, nenhum CSS imposto. Você recebe os dados e cria o layout exatamente como o design pede.

Infraestrutura própria

PostgreSQL + Redis + Node.js em Docker. Sem dependência de serviços terceiros de widget. Seus dados, sob seu controle.
